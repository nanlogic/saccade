# Saccade Public Evidence

This directory is for curated, sanitized, reproducible public evidence—not raw
development runs. Start with [`docs/PUBLIC_EVIDENCE_GUIDE.md`](../docs/PUBLIC_EVIDENCE_GUIDE.md).

Each published pack must contain a human-readable summary, structured report,
value-free replay, environment metadata, checksums, an uncut recording, and
web-ready preview media. Video and GIF assets illustrate a run; structured
same-WebView truth and verified native-input receipts determine the verdict.

Historical milestone files under `docs/` remain engineering records and may
describe obsolete builds. Public claims should point to a current pack here.
